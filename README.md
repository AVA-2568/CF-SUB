## CF-SUB
通过 Cloudflare Workers 搭建专属于你的订阅链接，目前测试下来应该没啥大毛病  
Page没测试，按理来说区别不大

### 可能用到的变量：

NODE_KV-----KV名称  
ADMIN_TOKEN-----你的登录密钥（建议不要低于15字符）   
SUB_TOKEN-----订阅密钥  

### 项目特点：

抛弃了订阅转换功能，完全纯粹的Base64,带给你更加安全的体验  
订阅和登录密钥分离，妈妈再也不用当心我后台被看光了  
后台添加拉取记录（匿名-只记录地区和次数）-测试中  
添加安全防护
